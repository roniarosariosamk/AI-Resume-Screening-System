# 📄 AI Resume Screening & Recruitment System

An AI-powered Resume Screening System built using Streamlit, LangChain, Ollama, Hugging Face Embeddings, and FAISS.

## 🚀 Features

- 📄 Upload multiple resumes (PDF)
- 📝 Upload Job Description
- 🤖 AI Resume Summary
- 🛠 Skill Extraction
- 📊 ATS Score Calculation
- 🏆 Resume Ranking
- 🔍 Candidate Search & Filter
- 📈 Analytics Dashboard
- 📥 CSV Report Generation
- 📄 PDF Report Generation
- 🎤 AI Interview Question Generator
- 🤖 AI Candidate Evaluation
- 📧 Automated Interview Invitation Email
- 💬 RAG-based Resume Chatbot
- 🔐 Admin Login

## 🛠 Technologies Used

- Python
- Streamlit
- LangChain
- Ollama (Llama 3)
- Hugging Face Embeddings
- FAISS
- Pandas
- Plotly
- ReportLab
